1. Pricing Calculator
This allows you to estimate the cost of your infrastructure, before you launch them.
Example, database and GKE

2. Cloud Deployment Manager
This is the Terraform for GCP, they use it for their IAC

3. Cloud Marketplace
This allows you to get commercial product that are not native to GCP, like Cassandra, Jenkins. See it like App store
for your phone.

4. Cloud DNS
This allows you to manage your domain, similar to route 53

5. Cloud Dataflow
This is a pipeline for data in GCP, It can process the data and send it to the final destination.

6. Cloud Dataproc
This is when you want some intelligence in your data process. GCP Spark or Hadoop. It uses VM
