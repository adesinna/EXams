Amazon Simple Storage Service (Amazon S3) is a highly scalable, secure,
and durable object storage service provided by AWS. It is designed to store
and retrieve any amount of data, from anywhere on the web, and is commonly
used for a variety of applications, including backup and recovery, data archiving, big data analytics, and content delivery.

Key Features of Amazon S3:

1. Object Storage:
        Data is stored as objects in buckets, with each object containing the
        data itself, metadata, and a unique identifier.

2. Scalability:
        S3 automatically scales to handle growing data needs without
        requiring user intervention.

3. Durability and Availability:
        S3 ensures high durability (99.999999999% or "11 nines") and availability,
        replicating data across multiple facilities within a region.

4. Security:
        Offers robust security features such as encryption at rest and in transit,
         access control policies, and integration with AWS Identity and Access Management (IAM).

5. Lifecycle Management:
        Supports lifecycle policies to transition data between storage classes or
        automatically delete it after a specified time.

6. Storage Classes:
    Offers various classes for different use cases, such as S3 Standard,
    S3 Intelligent-Tiering, S3 Glacier, and S3 One Zone-Infrequent Access.

7. Data Access and Transfer:
    Supports fine-grained access control via Access Control Lists (ACLs) and
    bucket policies, and allows direct integration with other AWS services.


 Common Use Cases:
- Hosting static websites
- Storing media files for applications
