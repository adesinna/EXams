Tools like Velero can help you take a backup of your cluster using the kubeapi server.

ETCD stores information about the cluster. It comes with a snapshot capability

ETCDCTL_API=3 etcdctl snapshot save snapshot.db

ETCDCTL_API=3 etcdctl snapshot status snapshot.db

ETCDCTL_API=3 etcdctl snapshot restore snapshot.db --data-dir /path of backup
